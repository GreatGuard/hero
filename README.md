# 英雄无敌 - 文字冒险游戏 (多语言版本)

这是一个Python文字冒险游戏，支持中文和英文两种语言，玩家扮演一位英雄在直线地图上进行冒险，游戏包含丰富的随机事件和成长系统。

## 游戏特点

1. **欢迎界面** - 游戏启动时显示游戏说明
2. **英雄系统** - 包含名字、等级、血量、攻击力、防御力、金币、经验值和技能
3. **成长系统** - 通过获得经验值升级，提升各项属性
4. **技能系统** - 随机学习或升级获得的技能，包括火球术、吸血、闪避等
5. **战斗系统** - 与怪物和Boss进行回合制战斗
6. **随机事件** - 多种随机事件：
   - 踩地雷/陷阱 - 掉血（受防御力影响）
   - 吃包子/发现泉水 - 恢复血量
   - 遇到怪物/强敌 - 战斗获得经验和金币
   - 发现宝箱 - 获得金币
   - 遇到商人 - 购买物品
   - 学习技能 - 获得新技能（已移除）
   - 发现药剂 - 增加药剂数量
7. **装备系统** - 武器、防具、饰品三种装备栏位，支持稀有度系统
8. **地图类型** - 平原、森林、沙漠、地牢、山脉、沼泽、雪原七种地图
9. **难度系统** - 简单、普通、困难、噩梦四个难度级别

## 新增功能 (v4.0)

1. **存档系统** - 支持多槽位存档和读档
2. **统计系统** - 追踪游戏数据和玩家表现
3. **任务系统** - 随机生成支线任务
4. **职业系统** - 三种职业选择（战士、法师、刺客）
5. **技能树系统** - 可视化技能升级路径
6. **装备强化与附魔** - 升级和附魔装备
7. **成就系统** - 解锁各种成就
8. **Boss战机制** - 多阶段Boss战斗
9. **游戏设置** - 自定义游戏体验
10. **游戏日志** - 记录游戏历史
11. **错误处理** - 增强游戏稳定性
12. **性能优化** - 提升游戏性能


## 游戏玩法

1. 选择语言（中文/英文）
2. 创建英雄并命名
3. 选择职业（战士/法师/刺客）
4. 选择地图类型和难度
5. 在新手村熟悉游戏机制和进行准备
6. 在地图上逐步前进（长度根据难度变化）
7. 每一步都可能触发随机事件
8. 通过战斗获得经验和金币
9. 升级提升属性并获得技能点
10. 使用技能点升级技能树中的技能
11. 使用药剂和技能在冒险中生存
12. 完成随机任务获得额外奖励
13. 击败Boss获得丰厚奖励
14. 到达终点即为胜利

## 新手村系统

- **训练场**：学习战斗技巧，获得初始经验
- **商店**：购买基础装备和物品（特惠价格）
- **诊所**：治疗伤口，恢复生命值
- **村长家**：获取游戏建议和指导
- **初始资金**：根据难度变化，20-30枚金币

## 地图类型

- **平原**：平坦的草原，适合新手，事件较温和
- **森林**：神秘的森林，有很多草药和野兽
- **沙漠**：炎热的沙漠，有危险的生物，脱水威胁
- **地牢**：阴森的地牢，强大的敌人，有鬼魂事件
- **山脉**：险峻的山脉，有巨人和龙，高风险高回报
- **沼泽**：有毒的沼泽，有特殊怪物和事件
- **雪原**：寒冷的雪原，有冰系怪物和特殊状态效果

## 难度系统

- **简单**：地图50格，金币/经验1.5倍，敌人0.7倍强度，30金币3药剂
- **普通**：地图100格，1倍奖励，1倍强度，20金币2药剂
- **困难**：地图150格，0.8倍金币1.2倍经验，敌人1.3倍强度，15金币1药剂
- **噩梦**：地图200格，0.6倍金币1.5倍经验，敌人1.6倍强度，10金币1药剂

## 装备系统

- **装备类型**：武器、防具、饰品
- **稀有度**：普通、优秀、稀有、史诗、传说（白绿蓝紫金）
- **属性加成**：攻击力、防御力、生命值
- **特殊属性**：暴击率、吸血、闪避、反击、元素伤害等
- **套装系统**：收集套装可获得额外加成
- **装备强化**：可强化装备提升属性（最高+15）
- **装备附魔**：可附魔装备获得特殊效果
- **背包系统**：可以收集和切换装备
- **装备商店**：随机生成商品，价格受稀有度影响

## 技能系统

### 基础技能
- **火球术**：战斗中可以释放强力魔法攻击，也能触发吸血和暴击
- **吸血**：每次攻击都会回复伤害值30%的血量
- **闪避**：20%概率完全避免伤害
- **暴击**：15%概率造成双倍伤害（火球术暴击为1.5倍）
- **治疗术**：战斗中可以恢复25-40点血量

### 新增技能
- **连斩**：连续攻击2次，每次造成50%伤害
- **护盾**：下次受到伤害减少50%
- **狂暴**：下3回合攻击提升50%，防御降低50%
- **专注**：下次攻击必中且暴击

### 职业技能
- **战士**：盾击、战吼
- **法师**：火球术、冰霜护甲
- **刺客**：背刺、影袭

### 技能树系统
- 每个职业有专属技能树
- 使用技能点升级技能
- 技能有前置要求和最大等级
- 提供可视化的技能升级界面

## 多语言支持

- 游戏启动时可以选择中文或英文界面
- 所有游戏文本都提供两种语言版本
- 随语言切换，怪物名称和游戏提示也会相应改变
